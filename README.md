# Sistema CRUD - Máquinas Expendedoras

## Descripción del Proyecto

Este proyecto consiste en el desarrollo de una API REST para la gestión de máquinas expendedoras, implementando operaciones CRUD (Crear, Leer, Actualizar y Eliminar).

La aplicación está desarrollada con:

- Node.js
- Express
- MySQL
- Docker

El sistema permite administrar información como:

- Nombre de la máquina
- Ubicación
- Capacidad
- Estado
